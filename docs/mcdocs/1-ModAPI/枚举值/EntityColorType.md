# EntityColorType

class in mod.common.minecraftEnum

- 描述

    描述实体的颜色种类，截止至网易2.9版本，该枚举值应用于羊和热带鱼的颜色



```python
class EntityColorType:
	White = 0  		# 白色
	Orange = 1  	# 橘色
	Magenta = 2  	# 品红
	LightBlue = 3  	# 浅蓝
	Yellow = 4 		# 黄色
	LightGreen = 5  # 浅绿
	Pink = 6  		# 粉色
	Gray = 7 		# 灰色
	Silver = 8  	# 银色
	Cyan = 9  		# 青色
	Purple = 10  	# 紫色
	Blue = 11  		# 蓝色
	Brown = 12 		# 棕色
	Green = 13  	# 绿色
	Red = 14  		# 红色
	Black = 15  	# 黑色

```
