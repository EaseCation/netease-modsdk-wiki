# 示例简介



## 示例下载

下载[2.3 Demo](https://g79.gdl.netease.com/2.3DemoV2.zip)。

下载后解压，示例DEMO位于“示例”文件夹。



## DemoMod

该文件夹内主要放置了ModSDK模组开发相关的示例

<span id="TutorialMod"></span>

* TutorialMod

  新手入门必备，展示了如何使用聊天信息发放物品
