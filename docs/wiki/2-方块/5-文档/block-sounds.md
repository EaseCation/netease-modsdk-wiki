# 方块声音

<!--@include: @/wiki/bedrock-wiki-mirror.md-->

方块声音指的是`blocks.json`中方块条目内的音效属性。  
该属性用于设置方块的基本音效，包含挖掘音效、踩踏音效、破坏音效和放置音效等。您可以通过以下方式为方块添加音效：

::: code-group
```json [RP/blocks.json]
{
  "format_version": [1, 1, 0],
  "wiki:custom_log": {
    "sound": "wood" // 在此定义音效
  }
}
```
:::
